# 📝 پنل ساخت آزمون معلم

یک Cloudflare Worker کامل برای ساخت آزمون‌های درسی با قابلیت خروجی PDF و Word.

## ✨ امکانات

- 📋 مدیریت اطلاعات آزمون (مدرسه، معلم، تاریخ، پایه و...)
- ✏️ انواع سوالات:
  - تشریحی
  - چهارگزینه‌ای
  - کوتاه‌پاسخ
  - صحیح و غلط
- 📊 پشتیبانی از بارم‌بندی (نمره‌دهی)
- 💬 بازخورد برای هر سوال
- 👁️ پیش‌نمایش زنده آزمون
- 📄 خروجی PDF با فرمت A4
- 📤 خروجی Word (DOC)
- 🧮 نمادهای ریاضی
- 📅 تاریخ شمسی (1405)
- 🌐 کاملاً فارسی و RTL

## 🚀 نحوه Deploy

### 1. کلون کردن ریپو
```bash
git clone https://github.com/naderuser/tosifapk.git
cd tosifapk
```

### 2. Deploy با Wrangler
```bash
npm install -g wrangler
wrangler login
wrangler deploy
```

### 3. تنظیم Custom Domain (اختیاری)
در داشبورد Cloudflare Workers می‌توانید یک دامنه سفارشی تنظیم کنید.

## 📱 نحوه استفاده

پس از Deploy، به آدرس Worker مراجعه کنید:

```
https://your-worker.your-subdomain.workers.dev
```

### افزودن سوال:
1. نوع سوال را انتخاب کنید
2. متن سوال را بنویسید
3. بارم (نمره) را وارد کنید
4. روی "افزودن سوال" کلیک کنید

### خروجی گرفتن:
- **PDF**: روی دکمه "PDF" کلیک کنید
- **Word**: روی دکمه "Word" کلیک کنید
